# 🏢 신도시 아파트 가격 예측
 
> 2기 신도시 개발 단계별 아파트 실거래가 데이터를 기반으로 머신러닝 모델을 활용하여 아파트 가격을 예측하는 데이터마이닝 프로젝트

 
## 📌 프로젝트 개요
 
신도시 개발이 아파트 가격에 미치는 영향을 분석하고, 다양한 변수를 활용하여 아파트 거래금액을 예측합니다.  
국토교통부 실거래가 공개시스템 데이터를 기반으로 피처 엔지니어링부터 모델 학습까지 전 과정을 수행합니다.

## 🔧 피처 엔지니어링
 
팀원별로 변수를 분담하여 원본 데이터에 추가하였습니다.
 
| 변수명 | 설명 | 수집 방법 |
|--------|------|-----------|
| `서울도심거리` | 광화문(서울 도심)까지의 거리 (km) | 카카오 로컬 API + Haversine 공식 |
| `단지별_세대수` | 아파트 단지의 총 세대수 | K-apt 공동주택 단지 기본정보 CSV |
| `발표후경과년수` | 신도시 발표 후 경과 연수 | 계약년도 기반 계산 |
| `자족용지비율` | 신도시 내 자족용지 비율 | 신도시별 계획 자료 |
| `지하철호선개수` | 인근 지하철 노선 수 | 교통 데이터 |
| `기차역까지의거리` | 가장 가까운 기차역까지의 거리 (km) | 교통 데이터 |
| `가장 가까운 지하철역까지의 거리` | 가장 가까운 지하철역까지의 거리 (km) | 교통 데이터 |
| `가장 가까운 IC와의 거리` | 가장 가까운 고속도로 IC까지의 거리 (km) | 교통 데이터 |

## 📊 KNN 모델 결과
 
**목표 변수**: `거래금액(만원)`
 
### 모델 학습 과정
1. k=5 기본 모델 학습 → 고가 구간 성능 저하 확인
2. GridSearchCV (k=1~10, 5-fold 교차검증)로 최적 k 탐색
3. 최적 k=3으로 최종 모델 학습
### 성능
 
| 지표 | k=5 (기본) | k=3 (최적) |
|------|-----------|-----------|
| RMSE | 7,295만원 | 7,341만원 |
| MAE | 4,136만원 | 4,123만원 |
 
### 피처 중요도
Permutation Importance 기반으로 `전용면적(㎡)`, `건축년도`, `서울도심거리` 순으로 중요도가 높게 나타남
